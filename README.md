# Clustering_Diseases
Taking headlines from a webscraped txt document, extracting location with diseases, applying kmeans for clustering and highlighting on a folium map
# Libraries used
matplotlib
numpy
scipy
pandas
sklearn
folium
geonamescache
unidecode
